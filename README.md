# Wintile

### Features
* Simple keyboard manipulation
* Easily Customize config

```
j, k     : Move focus
S-j, S-k : Move window
h, l     : Move window (when floating
S-h, S-l : Resize window
1~9      : Move virtual desktop
m        : Maximize
S-Enter  : Open terminal
r        : Launcher
S-d      : Close window
i        : Open browser
a        : Set as a main window
Space    : Change tiling mode
q        : Quit
```
